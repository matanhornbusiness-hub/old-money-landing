
# Old Money Landing — Ready for Vercel/Netlify

## מה בתוך ה־ZIP
- `index.html` — דף נחיתה RTL, מובייל־פרסט, כולל חשיפת שם המותג ב-HERO.
- `assets/` — תמונות דמה + `poster.jpg`. **שימו לב:** קובץ הווידאו `hero.mp4` לא קיים — תחליפו את הקובץ בעצמכם.
- `favicon.png` — איקון בסיסי (אפשר להחליף בלוגו).

## איך מעלים ל־Vercel (חינם)
1. התחברו ל־https://vercel.com/signup
2. לחצו New Project → Import באמצעות Drag&Drop של התיקייה כולה או העלאה דרך ה־Dashboard.
3. תקבלו כתובת חינמית כמו `https://YOUR-PROJECT.vercel.app`.
4. (אופציונלי) חברו דומיין משלכם ב־Vercel → Domains.

## איך מעלים ל־Netlify (חינם)
1. התחברו ל־https://www.netlify.com
2. ב־Sites → Drag & Drop the site folder — גררו את התיקייה כולה.
3. תקבלו כתובת חינמית כמו `https://YOUR-SITE.netlify.app`.

## איפה שמים את הווידאו
- הוסיפו קובץ `assets/hero.mp4` (8–12 שניות, H.264, ללא סאונד, < 5MB).
- יש Fallback לפוסטר `assets/poster.jpg` אם הווידאו לא נטען.

## החלפת טקסטים/מותג
- חפשו והחליפו `BRAND NAME` בשם המותג שלכם.
- עדכנו טקסטים/מחירים/מייל ליצירת קשר ב-`index.html`.

## טיפים לאיכות Old Money
- צבעי בסיס: שמנת (#F6F2E9), ירוק־יער (#0F3B2E), זהב חם (#B08D57), שחור עמוק (#0A0A0A).
- טיפוגרפיה: כותרות Serif, טקסטים Sans.
- צילום: טקסטורות בד, תקריבי זהב/פנינים, רקעים כהים, תאורה רכה.
